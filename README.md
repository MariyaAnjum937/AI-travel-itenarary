## âš™ï¸ What It Does

AI-Travel-Assistant takes three simple inputs:

1. **Destination City**
2. **Duration (in days)**
3. **Interests** (e.g., *Art Museums, Vegan Food, Hiking*)

It then generates a **day-wise itinerary** that includes:

* Morning, afternoon, and evening activity blocks
* Realistic durations and locations
* Grouped attractions to minimize travel time
* Suggestions for meals, leisure, and rest breaks

Example Output:

```
Day 1:
- Morning: Visit Art Museum (2 hrs) - Downtown
- Afternoon: Lunch at Green Eats (1 hr)
- Evening: Hike at Sunset Trail (3 hrs)
```
